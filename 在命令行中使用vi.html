由于再VSCode中使用了Vim，感觉棒极了。平时开发中也会需要使用命令行，每当需要移动、删除内容时，只能使用方向键移动，多按几次，感觉人都开始烦躁了。
 
幸好zsh命令行支持Vi模式😄

配置Vi

使用前提是安装了 zsh 命令行，一般默认的是 bin。然后还安装了 ohmyzsh
这两个搭配特别方便，推荐。
 
ohmyzsh 里面集成了很多插件，例如git、vi-mode等，具体可以看 ohmyzsh-plugins，因为默认自带有这个插件，所有只需要简单的配置一下就好了：
 
打开配置文件：
vim ~/.zshrc
 
添加vi-mode插件：
plugins=(git vi-mode)

## 添加下面命令可以动态提示 normal 或 insert 模式
VI_MODE_RESET_PROMPT_ON_MODE_CHANGE=true
可以参考下述截图
然后保存文件并提出，重开命令行工具或使用下面的命令使得配置文件立即生效：
source ~/.zshrc
 

使用

默认是 insert 模式，搞了一个脑图示意相关命令，详细命令可以查看 vi-mode plugin
 
 

待研究

虽然用起来真的是爽了，但还有些值得探索的：
  1. insert模式下支持快捷键的左右移动
      2. 可以修改 .zshrc 文件，手动绑定key，使用 ctrl + f (往前/往右移动)、ctrl + b（往后/往左移动）：
      bindkey '^f' forward-char
bindkey '^b' backward-char
  1. insert模式是否支持删除到行尾
      2. 同上，自定义绑定一个key，使用 ctrl + k 删除到当前光标到行尾
      bindkey '^k' kill-line
       
更多命令可以参考：
